## 💡 Customization Guide

### Make It More Dynamic
```jsx
// Increase orb movement
y: [0, 50, 0]  // Was [0, 30, 0]
x: [0, 25, 0]  // Was [0, 15, 0]

// Increase particle count
{[...Array(40)]}  // Was 20

// Increase spotlight intensity
opacity: [0.8, 1.0, 0.8]  // Was [0.6, 0.8, 0.6]
```

### Make It More Subtle
```jsx
// Decrease orb movement
y: [0, 15, 0]  // Was [0, 30, 0]

// Decrease particle count
{[...Array(10)]}  // Was 20

// Reduce spotlight intensity
opacity: [0.4, 0.6, 0.4]  // Was [0.6, 0.8, 0.6]
```

### Adjust Speed
```jsx
// Slower (more elegant)
duration: 15  // Was 12
duration: 20  // Was 15
duration: 25  // Was 18

// Faster (more dynamic)
duration: 8   // Was 12
duration: 10  // Was 15
```
